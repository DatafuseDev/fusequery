---
title: NOW
---

Returns the current date and time.

## Syntax

```sql
NOW()
```

## Return Type

Datetime object, returns date and time in “YYYY-MM-DD hh:mm:ss” format.

## Examples

```sql
SELECT NOW();
+---------------------+
| NOW()               |
+---------------------+
| 2021-09-03 01:26:07 |
+---------------------+
```
