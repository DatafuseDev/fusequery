title: VERSION
---

Return the current version information of DatafuseQuery.

## Syntax

```
SELECT VERSION()
```

## Examples

```
mysql> SELECT VERSION();
+----------------------------------------------------------------------------------------+
| version()                                                                              |
+----------------------------------------------------------------------------------------+
| DatafuseQuery v-0.1.0-0f9ec31-simd(1.56.0-nightly-2021-08-10T15:25:36.875868571+00:00) |
+----------------------------------------------------------------------------------------+
```
