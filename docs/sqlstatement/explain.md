title: EXPLAIN
---

Show the execution plan of a statement.

Syntax:
```text
EXPLAIN [GRAPH | PIPELINE] SELECT ...
```

EXPLAIN Types
* PLAN — Query execution plan.
* GRAPH — Query execution plan with graphviz.
* PIPELINE — Query execution pipeline.


## Explain

```text
mysql> EXPLAIN SELECT sum(number+3)/count(number) FROM system.numbers(100000);
+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| explain                                                                                                                                                                         |
+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Aggregate: groupBy=[[]], aggr=[[(sum([(number + 3)]) / count([number]))]]
  ReadDataSource: scan parts [8](Read from system.numbers table, Read Rows:100000, Read Bytes:800000) |
+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
1 row in set (0.00 sec)
```

## Explain PIPELINE

```text
mysql> EXPLAIN PIPELINE SELECT sum(number+3)/count(number) FROM system.numbers(100000);
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| explain                                                                                                                                                                                                                                      |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| 
  └─ AggregateFinalTransform × 1 processor
    └─ Merge (AggregatePartialTransform × 8 processors) to (MergeProcessor × 1)
      └─ AggregatePartialTransform × 8 processors
        └─ SourceTransform × 8 processors                      |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
1 row in set (0.00 sec)
```
